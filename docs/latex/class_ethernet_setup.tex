\doxysubsection{Detailed Description}
Setting up an T\+C\+P/\+U\+DP Ethernet connection. 

Stores and provides acces to the Server and U\+DP instance currently setup. Once the setup is done for a given \mbox{\hyperlink{class_network_interface}{Network\+Interface}} the instance of this class is not needed anymore. An instance will be created within the Networ\+Interface setup function ans go out of scope after the setup has been done.
