\doxysubsection{Detailed Description}
Main entry point used to instatiate the network interface(s). 

The member functions of this class shall be the only ones used in the main Comand\+Station-\/\+EX sketch. It provides the setup and callback setting facilities as well as the loop managing data transfer and command handling accroding to the Command\+Station command protocols available.
