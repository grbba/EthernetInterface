Set the App Protocol. 

The detection id done upon the very first message recieved. The client will then be bound to that protocol. Its very brittle as e.\+g. The N message as first message for Wi\+Throttle is not a requirement by the protocol; If any client talking Withrottle doesn\textquotesingle{}t implement this the detection will default to J\+M\+RI. For H\+T\+TP we base this only on a subset of th H\+T\+TP verbs which can be used.
