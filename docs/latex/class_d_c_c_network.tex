\doxysubsection{Detailed Description}
Core class holding and running the instantiated \mbox{\hyperlink{class_transport}{Transport}} objects initalized through the \mbox{\hyperlink{class_network_interface}{Network\+Interface}}. 

The Transports are handled as abstract transports as they are derived from a templated class. \mbox{\hyperlink{class_transport}{Transport}} inherits from \mbox{\hyperlink{struct_abstract_transport}{Abstract\+Transport}}. The transports arrayactually holds the instatiated \mbox{\hyperlink{class_transport}{Transport}} instances. The number of transports is limited by M\+A\+X\+\_\+\+I\+N\+T\+E\+R\+F\+A\+C\+ES
