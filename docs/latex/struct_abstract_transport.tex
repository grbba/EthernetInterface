\doxysubsection{Detailed Description}
Abstract parent class of the templated ( Ethernet or Wifi ) \mbox{\hyperlink{class_transport}{Transport}} class. 

Instances of \mbox{\hyperlink{class_transport}{Transport}} are hold through this in an array in \mbox{\hyperlink{class_d_c_c_network}{D\+C\+C\+Network}} which describes and actually manages the available transports.
