bool Transport< S, C, U >::setup ( NetworkInterface *  nw)

Initalizes either the TCP or WiFi transport and creates the connecion pool.

If the port for this transport is 23 and the protocol TCP it will start also the telnet server and set the maxConnections to 1 ( only one telnet connection will be allowed )
