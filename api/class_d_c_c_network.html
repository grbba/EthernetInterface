class DCCNetwork¶

Core class holding and running the instantiated Transport objects initalized through the NetworkInterface. The Transports are handled as abstract transports as they are derived from a templated class. Transport inherits from AbstractTransport. The transports arrayactually holds the instatiated Transport instances. The number of transports is limited by MAX_INTERFACES
